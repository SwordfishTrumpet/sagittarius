# Screenshots

This directory contains UI screenshots for the README and documentation.

## Required Screenshots

To complete the visual documentation, capture the following:

| Filename | Description | Dimensions |
|----------|-------------|------------|
| `inbox-three-pane.png` | Main three-pane interface showing sidebar, message list, and reading pane | 1440x900 |
| `compose-rich-text.png` | Composer with formatting toolbar and attachment area | 1440x900 |
| `thread-conversation.png` | Expanded thread view with multiple messages | 1440x900 |
| `mobile-responsive.png` | Mobile layout (sidebar collapsed) | 375x812 |
| `settings-filters.png` | Sieve filter editor in settings | 1440x900 |
| `dark-mode.png` | Dark theme variant (if applicable) | 1440x900 |

## Guidelines

1. Use consistent test data (not personal emails)
2. Capture at 100% browser zoom
3. Use a clean browser (no extensions visible)
4. macOS: Use **CleanShot X** or **Cmd+Shift+5** for window capture
5. Linux: Use **Flameshot** or **GNOME Screenshot**
6. Windows: Use **Win+Shift+S** or **Snipping Tool**
